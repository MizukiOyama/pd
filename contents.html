Maine Service

メインサービス

    HP制作

  1. コーディング：10万～20万

    作業内容：基本的なHTML/CSS/JavaScriptを使用したコーディング。「TOP、About、Service、Contact、Information」などの1～5ページのシンプルなサイトを制作いたします。

  2. ノーコード：10万～20万

    作業内容：WordPressやSTUDIOなどのノーコードツールでシンプルなサイトを作成いたします。テーマやプラグインの設定を行い、基本的なカスタマイズしたサイトを提供します。

  3. コーディング「テンプレート」：8万～15万

    作業内容：既存のテンプレートを使用して、少しのカスタマイズや編集を行い、クライアントの要望に沿ったサイトを作成いたします。

  4. ノーコード「テンプレート」：5万～15万

    作業内容：テンプレートを使用し、簡単なカスタマイズやコンテンツの差し替えを行う。ノーコードツールを使用することで、制作コストを抑えたサイトを提供します。

    5. LP制作

  5. コーディング：10万～15万

    作業内容：シンプルな1ページランディングページのコーディング。HTML/CSS/JavaScriptを使用し、デザインを重視したLP制作を行います。

  6. ノーコード：10万～15万

    作業内容：WordPressやSTUDIOなどのノーコードツールで作成。テンプレートを活用しつつ、クライアントのブランドに合わせたカスタマイズを行います。

  7. コーディング「テンプレート」：8万～12万

    作業内容：テンプレートを使用したLP制作。デザインやコンテンツのカスタマイズを行い、コストを抑えたサイトを制作いたします。

  8. ノーコード「テンプレート」：5万～10万

    作業内容：テンプレートをベースに、簡単なカスタマイズやコンテンツの配置を行い、短期間でのLP制作が可能です。

    Web Design

  1. オウンドメディア：10万～20万

    作業内容：クライアントが自らコンテンツを発信するためのブログやニュースサイトなどをデザイン。主にシンプルで機能的なデザインを提供し、初期段階のSEO設計も含む。

  2. コーポレートサイト：10万～15万

    作業内容：企業や団体向けのシンプルな会社概要ページやサービス紹介サイトのデザインを行います。企業イメージを反映させたビジュアル「基本的なレイアウト、ブランディング」に重点を置きデザインを制作いたします。

  3. ブログサイト：8万～12万

    作業内容：ブログ専用サイトのデザイン。シンプルなレイアウトで、ユーザビリティやSEOを考慮したブログページのデザインを行います。

    4. Coding

  4. HTML, CSS, JS (jQuery)：10万～15万

    作業内容：HTML/CSS/JavaScript（jQueryを含む）を使用して、静的なウェブサイトやアニメーションなどの基本的な実装。小規模なサイトや特定の機能に絞ったシンプルなコーディングを行います。

    5. HPリニューアル

  5. 自社制作サイト：8万～12万

    作業内容：自分で制作したサイトのデザインや機能を改善するリニューアル作業。既存サイトの改善にフォーカスし、UI/UXの向上や最新のトレンドに合わせた変更を行い、新しいサイトをご提供いたします。

Concept

サイトコンセプト

当サイトは、シンプルで先錬されたデザインで、優しそうな人物をコンセプトに制作しました。 白と黒を基調とし、落ち着いた大人の雰囲気を演出しつつ、フォント選びにこだわり、柔らかさと温かみを表現しています。 無駄を削ぎ落としたシンプルなデザインに、クラシックな要素を取り入れることで、 時代を超えた上品さを感じさせる仕上がりとなっています。
